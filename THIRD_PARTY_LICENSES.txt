Third-Party License Summary for lingopilot-tts-piper
====================================================

Project license context
-----------------------

`lingopilot-tts-piper` is distributed under GPL-3.0-only. In this repository,
the binary links the vendored `espeak-rs-sys` crate, which wraps eSpeak-NG.
That linkage determines the binary's GPL-3.0-only distribution posture.

This file is a repository-authored summary for release packaging and operator
visibility. It does not replace the original upstream license texts.

   Role: phonemization runtime used by Piper through `espeak-rs` and `espeak-rs-sys`
